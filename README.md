# CODE-A-NOVA Workspace

Welcome to the CODE-A-NOVA repository! This workspace contains three distinct, fully-featured full-stack applications built to solve real-world problems.

## 🚀 The Projects

### 1. FestFlow
**FestFlow** is a comprehensive Event Management System designed to handle university or college fests. It provides seamless registration for students, schedule tracking, and an admin dashboard to manage events and participants efficiently. 

### 2. HireBoard
**HireBoard** is a modern Job Application and Recruitment platform. It bridges the gap between employers and job seekers, featuring dedicated dashboards for candidates to manage their applications and saved jobs, and for employers to post jobs and review applicants. 

### 3. StockPilot
**StockPilot** is an intuitive Inventory and Stock Management System. It helps businesses track their products, manage suppliers, monitor low stock alerts, and analyze sales and purchases through a clean, data-driven dashboard.

---

## 🛠️ How We Made It

All three projects were built with a robust and modern tech stack, ensuring scalability, performance, and a stunning user experience:

- **Frontend (Client)**: 
  - Powered by **React** and bundled with **Vite** for lightning-fast development.
  - Styled with **Tailwind CSS** for responsive, beautiful, and dynamic user interfaces.
  - State and routing managed natively with React Hooks and React Router.
- **Backend (Server)**:
  - Built on **Node.js** and **Express.js** to provide scalable RESTful APIs.
  - Data stored and queried using **MongoDB** (with Mongoose ODM).
  - Secure authentication flows implemented using **JWT** (JSON Web Tokens) and bcrypt.
- **Deployment Ready**:
  - The architectures are specifically configured to be deployed effortlessly to **Vercel** (for the frontends) and **Railway** (for the backends), utilizing dynamic environment variables for API integrations.
